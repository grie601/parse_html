Алгоритм работы 
1.Обработать HTML страницу - Каждый тэг сделать с новой строки. Для удобсвта парсинга
2. Выбрать вест текст находящийся между тэгами "body" для отсеивания точно не нужного текста
3. Получаем весь текст, который находится между выбранных тэгов. В алогритме выбран только тэг "p", но можно выбрать любой
4. В полученном тексте все ссылки редактируем по заданию, и при помощи регулярных выражений удаляем ненужные символы
5. Очищаем оставшийся текст от тэгов, которые остались в тексте
6. Из полученного url получаем строку с путем, которую соединяем с базовой директорией и получаем директорию для установки файла
7. Записываем файл по полученному пути

Запуск:
	Запуск программы осуществляется через командную строку CUR_DIR>python parse_url.py с параметром -u (либо -url). Результат будет находится в папке с файлом скрипта в директории в соответсвии с выбранным url
